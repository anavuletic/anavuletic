This project was meant to be explored by the user to discover what the room was capable of doing. We intended for there to be no instructions, but have the experience easy to use. The room is immersive as the user feels alone in a dark room but has only 2 objects they need to interact with. The room embodies a dream-like state where the user feels like they are in an alternate, bizarre reality.
